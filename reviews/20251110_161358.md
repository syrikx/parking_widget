# Parking Widget - 위젯 크기 최소화

**생성 시간**: 2025-11-10 16:13:58

## 요구사항

parking_widget 최소 크기를 1x1 또는 1x2부터 가능하게 간략화

## 변경 요약

- 파일 생성: 0개
- 파일 수정: 3개
- 파일 삭제: 0개

## 상세 변경사항

### 1. android/app/src/main/res/xml/parking_widget_info.xml

**작업**: 파일 수정
**이유**: Android 위젯 최소 크기를 3x2에서 1x1로 변경 (40dp = 약 1칸)

**변경 전:**
```
    android:minWidth="180dp"
    android:minHeight="110dp" 
```

**변경 후:**
```
    android:minWidth="40dp"
    android:minHeight="40dp" 
```
